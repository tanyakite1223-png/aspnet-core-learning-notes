# ASP.NET Core MVC 筆記：Razor 語法基礎 — `@`、`@{ }`、`@if`、`@foreach`

## Razor 是什麼？

Razor 是 ASP.NET Core MVC 的 **View Engine（視圖引擎）**，讓你在 HTML 裡面插入 C# 程式碼。

- Razor 檔案的副檔名是 `.cshtml`
- 執行時會把 C# 的部分換成實際的值，最後產生**純 HTML** 送給瀏覽器
- 這個動作叫做 **render（渲染）**

> 瀏覽器只看到 HTML，看不到任何 `@` 符號或 C# 程式碼。

---

## 核心觀念：模式切換

Razor 在兩種模式之間自動切換：

```
HTML 模式（預設）
  │
  └─ 遇到 @ → 切換到 C# 模式
       │
       ├─ @變數 → 印出值，切回 HTML
       ├─ @(表達式) → 算出結果印出，切回 HTML
       ├─ @{ ... } → 進入 C# 區塊（不輸出）
       └─ @if / @for / @foreach { ... }
            │
            └─ 大括號內預設是 C# 模式
                 │
                 └─ 遇到 <HTML 標籤> → 切回 HTML 模式
                      └─ 遇到 </HTML 標籤> → 切回 C# 模式
```

**一句話記住：`@` 是從 HTML 進入 C# 的開關，`<HTML 標籤>` 是從 C# 回到 HTML 的開關。**

---

## 語法一覽表

| 語法 | 用途 | 範例 |
|------|------|------|
| `@變數名` | 印出變數的值 | `@userName` |
| `@(表達式)` | 印出運算結果 / 明確劃定變數邊界 | `@(age + 1)`、`@(userName)安安` |
| `@{ ... }` | C# 程式碼區塊（執行但不輸出） | `@{ int x = 0; }` |
| `@* ... *@` | Razor 註解（不會出現在 HTML） | `@* 這是註解 *@` |
| `@if / else` | 條件判斷 | `@if (age >= 18) { <p>成年</p> }` |
| `@for` | 有 index 的迴圈 | `@for (int i = 0; i < arr.Length; i++)` |
| `@foreach` | 直接取值的迴圈 | `@foreach (var item in arr)` |

---

## `@變數名`：印出變數的值

```cshtml
@{
    string userName = "Amber";
    int age = 30;
}

<h1>歡迎，@userName！</h1>
<p>妳今年 @age 歲。</p>
```

輸出的 HTML：
```html
<h1>歡迎，Amber！</h1>
<p>妳今年 30 歲。</p>
```

### Razor 怎麼判斷變數名的結束？

Razor 從 `@` 開始往後讀，遇到**不可能是變數名的字元**就停下來：

| 字元類型 | 會停下來嗎？ | 範例 |
|---------|-----------|------|
| 空格 | ✅ 會 | `@userName 安安` → Amber 安安 |
| 標點符號（`!`、`,`、`.` 等） | ✅ 會 | `@userName!` → Amber! |
| HTML 的 `<` | ✅ 會 | `@userName</h1>` → Amber |
| 中文字 | ❌ 不會 | `@userName安安` → ❌ 錯誤（變成找 `userName安安`） |
| 英文字母、數字 | ❌ 不會 | `@userName123` → ❌ 錯誤 |

> C# 允許用中文當變數名，所以 Razor 會把中文字也讀進變數名裡。

## `@(表達式)`：小括號的兩個用途

### 用途 1：做運算

```cshtml
<p>明年妳就 @(age + 1) 歲了！</p>   @* ✅ 輸出 31 *@
<p>明年妳就 @age + 1 歲了！</p>     @* ❌ 輸出 30 + 1（+ 1 變純文字） *@
```

### 用途 2：明確劃定變數邊界

```cshtml
<h1>歡迎，@(userName)安安</h1>      @* ✅ 輸出 Amber安安 *@
<h1>歡迎，@userName安安</h1>        @* ❌ 找不到 userName安安 這個變數 *@
```

**核心觀念：小括號 `()` 告訴 Razor「這整段都是 C#，到右括號為止」。**

---

## `@{ ... }`：程式碼區塊

在 `@{ }` 裡面可以寫任意的 C# 程式碼，但**不會在頁面上輸出任何東西**。

```cshtml
@{
    string userName = "Amber";
    int age = 30;
    string city = "台中";
}
```

通常用來：
- 宣告變數
- 做計算（先算好，後面再用 `@變數` 印出來）
- 寫邏輯

---

## `@* ... *@`：Razor 註解

```cshtml
@* 這段註解不會出現在 HTML 裡 *@
```

### 三種註解的比較

| 寫法 | 寫在哪 | 會出現在最後的 HTML 嗎？ |
|------|--------|----------------------|
| `// 註解` | C# 程式碼區塊裡（`@{ }` 裡面） | ❌ 不會 |
| `<!-- 註解 -->` | HTML 裡 | ✅ 會（瀏覽器看原始碼看得到） |
| `@* 註解 *@` | Razor 檔案任何地方 | ❌ 不會 |

> 如果註解**不想讓使用者看到**，用 `@* *@`，不要用 `<!-- -->`。

## `@if / else`：條件判斷

```cshtml
@if (age >= 18)
{
    <p>妳是成年人，可以進入。</p>
}
else
{
    <p>未成年，請由家長陪同。</p>
}
```

### 重點：大括號裡面的模式

`@if` 的大括號裡面，Razor **預設是 C# 模式**：
- 遇到 `<HTML 標籤>` → 自動切換成 HTML 模式，輸出內容
- 純文字（沒有 HTML 標籤包裹）→ Razor 當作 C# 程式碼 → **編譯錯誤**

### 常見錯誤

```cshtml
@* ❌ 錯誤：純文字在 C# 區塊裡，Razor 當成 C# 處理 *@
@if (age >= 18)
{
    妳是成年人，可以進入。
}

@* ✅ 正確：用 HTML 標籤包裹 *@
@if (age >= 18)
{
    <p>妳是成年人，可以進入。</p>
}
```

如果真的需要在 C# 區塊裡輸出純文字（不包 HTML 標籤）：
- `<text>純文字</text>`：Razor 專用假標籤，輸出時 `<text>` 標籤會被移除
- `@:純文字`：單行純文字標記

實務上很少用，因為文字通常都會包在 HTML 標籤裡。

## `@for`：有 index 的迴圈

```cshtml
@{
    string[] fruits = { "蘋果", "香蕉", "葡萄", "芒果" };
}

@for (int i = 0; i < fruits.Length; i++)
{
    <p>@(i + 1). @fruits[i]</p>
}
```

輸出：
```
1. 蘋果
2. 香蕉
3. 葡萄
4. 芒果
```

---

## `@foreach`：直接取值的迴圈

```cshtml
@foreach (var item in fruits)
{
    <p>@item</p>
}
```

輸出：
```
蘋果
香蕉
葡萄
芒果
```

### `for` vs `foreach` 的差別

| | `for` | `foreach` |
|---|---|---|
| 給你什麼 | **index（編號）** | **元素本身（值）** |
| 怎麼拿值 | 用 index 查：`fruits[i]` | 直接就是值：`item` |
| 什麼時候用 | 需要 index | 只需要值 |

> **需要 index → 用 `for`。只需要值 → 用 `foreach`。**

---

## `@foreach` + `@if`：篩選 + 迴圈的組合

```cshtml
@{
    string[] fruits = { "蘋果", "香蕉", "葡萄", "芒果" };
    int num = 0;
}

@for (int x = 0; x < fruits.Length; x++)
{
    if (fruits[x].Contains("果"))
    {
        num++;
        <p>@num. @fruits[x]</p>
    }
}
```

輸出（只有含「果」的水果）：
```
1. 蘋果
2. 芒果
```

---

## 常見錯誤

### ❌ 運算沒用小括號

```cshtml
@* ❌ + 1 變成純文字 *@
<p>明年 @age + 1 歲</p>        @* 輸出：明年 30 + 1 歲 *@

@* ✅ 用小括號包起來 *@
<p>明年 @(age + 1) 歲</p>      @* 輸出：明年 31 歲 *@
```

### ❌ 變數後面直接接中文

```cshtml
@* ❌ Razor 把中文也讀進變數名 *@
<h1>@userName安安</h1>          @* 錯誤：找不到 userName安安 *@

@* ✅ 用小括號劃定邊界 *@
<h1>@(userName)安安</h1>        @* 輸出：Amber安安 *@
```

### ❌ 在 C# 區塊放純文字（沒有 HTML 標籤）

```cshtml
@* ❌ 純文字被當作 C# 程式碼 *@
@if (age >= 18)
{
    妳是成年人
}

@* ✅ 用 HTML 標籤包裹 *@
@if (age >= 18)
{
    <p>妳是成年人</p>
}
```
